# Suggested Commands for ORY-VAULT DMS

## Development Workflow
- `make init`: Create directory structure.
- `make up`: Start all services via docker-compose.
- `make down`: Stop all services.
- `make migrate`: Run Ory Kratos and Keto migrations.
- `make status`: Check health status.
- `make clean`: Total cleanup (removes volumes).
- `make logs`: Follow all logs.
- `make check`: Quick health check.

## System Commands (Windows)
- Use `ping` to check domains.
- Use `docker exec` for container interactions.
- Use `PowerShell` for general scripting.
