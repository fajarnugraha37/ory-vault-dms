# Specification: app/globals.css

## Overview
Global CSS/Tailwind configuration defining the design system variables and global utility classes.

## Functionality
*   **Theme:**
    *   Dark-themed background palette (`background-deep`, `background-base`, `background-elevated`).
    *   Accent color palette (`#5E6AD2`).
*   **Utilities:**
    *   `.bg-grid`: Tech-themed background grid.
    *   `.bg-noise`: Grainy texture overlay.
    *   `.text-gradient`: Gradient text styling.
    *   `.shadow-linear-card`: Card shadows.
