# Specification: ui/sonner.tsx

## Overview
Toast notification wrapper for `sonner`.

## Functionality
*   **User Interface:**
    *   Toaster component configured with project-specific icons and theming.

## Logic & Data Handling
*   **State Management:**
    *   Managed by `sonner`.

## Dependencies
*   `sonner`: Notification library.
*   `next-themes`: Theme awareness.
*   `lucide-react`: Icons.
