# Specification: ui/scroll-area.tsx

## Overview
Scroll area component using Base UI primitives for custom scrollbars.

## Functionality
*   **User Interface:**
    *   Scrollable viewport with custom scrollbar track and thumb.

## Logic & Data Handling
*   **State Management:**
    *   Handled by Base UI `ScrollAreaPrimitive`.

## Dependencies
*   `@base-ui/react/scroll-area`: Base primitives.
*   `lib/utils`: `cn` utility.
