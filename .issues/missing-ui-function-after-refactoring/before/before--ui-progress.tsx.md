# Specification: ui/progress.tsx

## Overview
Progress bar component using Base UI `Progress` primitive.

## Functionality
*   **User Interface:**
    *   Track and indicator progress bar.
    *   Optional label and percentage value display.

## Behavior
*   Uses Base UI `Progress` to manage bar state.

## Dependencies
*   `@base-ui/react/progress`: Base primitives.
*   `lib/utils`: `cn` utility.
