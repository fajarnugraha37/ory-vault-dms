# Specification: components/ui/dropdown-menu.tsx

## Overview
A lightweight dropdown menu component set based on Radix UI primitives.

## Functionality
*   **UI:** 
    *   Provides standard menu structure: `DropdownMenu`, `Trigger`, `Content`, `Item`, `Portal`.
    *   Optimized for consistent theme styling (blur, borders).
*   **Navigation:**
    *   Dropdown interaction.

## Behavior
*   Uses `Radix UI` for accessibility.
*   Includes entry animations.

## Dependencies
*   `@radix-ui/react-dropdown-menu`.
*   `lucide-react` (icons, though current code doesn't use the imported ones).
*   `lib/utils.ts`.

## Potential Issues
*   None.
