# Specification: lib/utils.ts

## Overview
Shared utility functions for class name merging (`cn`) and data formatting.

## Functionality
*   **Utility:**
    *   `cn`: Combines `clsx` and `tailwind-merge` for robust Tailwind class merging.
    *   `formatBytes`: Converts bytes to human-readable format.

## Behavior
*   Standard utility pattern.

## Dependencies
*   `clsx`.
*   `tailwind-merge`.

## Potential Issues
*   None.
