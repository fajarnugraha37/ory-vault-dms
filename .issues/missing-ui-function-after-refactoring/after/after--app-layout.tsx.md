# Specification: app/layout.tsx

## Overview
Root layout configuration for the application.

## Functionality
*   **Structure:**
    *   Sets up global fonts (`Geist`, `Geist_Mono`).
    *   Wraps application with `SceneWrapper`.
    *   Initializes `Toaster` (sonner).
*   **Metadata:**
    *   Sets document title and meta-description.

## Behavior
*   Forced "dark" mode via `html` class.

## Potential Issues
*   None.
