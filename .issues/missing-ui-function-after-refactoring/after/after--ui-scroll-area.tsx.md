# Specification: components/ui/scroll-area.tsx

## Overview
A custom scroll area component built on `@base-ui/react/scroll-area`.

## Functionality
*   **UI:** 
    *   Viewport container.
    *   Custom scrollbars with thumb.
*   **API:**
    *   Supports orientation (`vertical` default).

## Behavior
*   Uses Base UI primitives to ensure consistent cross-browser scrolling and accessibility.

## Dependencies
*   `@base-ui/react/scroll-area`.
*   `lib/utils.ts`.

## Potential Issues
*   None.
