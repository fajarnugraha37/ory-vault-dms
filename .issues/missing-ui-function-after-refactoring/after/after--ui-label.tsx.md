# Specification: components/ui/label.tsx

## Overview
A standard Label component to be used with form inputs.

## Functionality
*   **UI:**
    *   Flex-based layout with gap support.
    *   Handles peer-disabled state.
*   **API:**
    *   Standard HTML label attributes.

## Behavior
*   Uses `data-slot="label"` for internal identification.

## Dependencies
*   `lib/utils.ts`.

## Potential Issues
*   None.
