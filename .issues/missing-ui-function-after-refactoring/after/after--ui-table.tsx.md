# Specification: components/ui/table.tsx

## Overview
A compositional Table component set for structured data display.

## Functionality
*   **UI:**
    *   `Table` (with overflow container), `TableHeader`, `TableBody`, `TableFooter`, `TableRow`, `TableHead`, `TableCell`, `TableCaption`.
*   **Behavior:**
    *   `Table` auto-wraps in a `div` for responsiveness.

## Dependencies
*   `lib/utils.ts`.

## Potential Issues
